# PACIXP Website

The static marketing website for **Pacific Islands Internet Exchange Incorporated (PACIXP)** — a New Zealand-registered non-profit establishing internet exchange points (IXPs) across Pacific Island nations to reduce latency, lower transit costs, and strengthen regional digital sovereignty.

## About PACIXP

Pacific Island nations currently route internet traffic through distant hubs in Australia, Hawaii, or North America — even for communication between neighbouring islands. PACIXP addresses this by establishing distributed IXPs, beginning with **Samoa and Fiji**, to enable direct local peering.

**Projected impact:**
- Latency: 180–250ms → **< 80ms**
- Traffic localization: **> 35%** kept within the region
- Transit cost savings: **$1.2M–$1.8M USD annually**

Supported by APNIC, ISOC, Internet Association of Australia, and the New Zealand Internet Exchange (NZIX).
